# Σύστημα Διαχείρισης ΜΑΠ (Μέσα Ατομικής Προστασίας)

Αυτό το αποθετήριο περιέχει τον πηγαίο κώδικα για ένα ψηφιακό σύστημα διαχείρισης και χορήγησης Μέσων Ατομικής Προστασίας (ΜΑΠ) σε εργαζόμενους μιας βιομηχανίας.

## Λειτουργίες Εφαρμογής
* **Προβολή Αποθέματος:** Δυνατότητα προβολής των διαθέσιμων ΜΑΠ στην αποθήκη.
* **Καταχώρηση Χορήγησης:** Φόρμα επιλογής εργαζομένου και εξοπλισμού με αυτόματη μείωση του αποθέματος.
* **Ιστορικό Χορηγήσεων:** Προβολή αναλυτικού ιστορικού διανομής εξοπλισμού.

## Τεχνολογίες που χρησιμοποιήθηκαν
* HTML/CSS (Διεπαφή Χρήστη)
* PHP (Λογική Εφαρμογής & Επικοινωνία με Βάση)
* MySQL (Βάση Δεδομένων)

*Σημείωση: Το project αναπτύχθηκε στα πλαίσια ακαδημαϊκής εργασίας.*
