# Σύστημα Διαχείρισης ΜΑΠ & Σχεδιασμός Εταιρικού Δικτύου

Αυτό το αποθετήριο αποτελεί το τελικό παραδοτέο για το εργαστηριακό project: "Τεχνολογίες Διαδικτύου στην Ψηφιακή Βιομηχανία".

## 1. Περιγραφή Σεναρίου
Η εφαρμογή αναπτύχθηκε για τις ανάγκες μιας φανταστικής βιομηχανικής εγκατάστασης. Το σύστημα αναλαμβάνει την ψηφιακή διαχείριση της αποθήκης όσον αφορά τα Μέσα Ατομικής Προστασίας (ΜΑΠ). Ο υπεύθυνος ασφαλείας μπορεί να ελέγχει το διαθέσιμο απόθεμα, να χορηγεί εξοπλισμό στους εργαζομένους και να διατηρεί ψηφιακό ιστορικό όλων των διανομών.

## 2. Αρχιτεκτονική Εφαρμογής & Διαδικασία Ανάπτυξης
Η διαδικτυακή εφαρμογή αναπτύχθηκε τοπικά χρησιμοποιώντας τον Web Server Apache (XAMPP). 
Τα βήματα υλοποίησης περιλάμβαναν:
* Σχεδιασμό της λογικής και του UI με HTML, CSS και PHP.
* Δημιουργία αρχείου σύνδεσης με τη βάση δεδομένων.
* Ανάπτυξη τριών βασικών λειτουργικών σελίδων και ενός κεντρικού μενού πλοήγησης.

## 3. Δομή της Βάσης Δεδομένων
Χρησιμοποιήθηκε βάση δεδομένων MySQL με το όνομα `map_system`. Αποτελείται από 3 πίνακες:
1. `employees`: Αποθήκευση στοιχείων προσωπικού (Ονοματεπώνυμο, Ρόλος).
2. `ppe_items`: Διαχείριση των διαθέσιμων ΜΑΠ και της ποσότητάς τους.
3. `issuances`: Πίνακας ιστορικού που ενώνει τους εργαζόμενους με τα αντικείμενα.

## 4. Χρήση Εργαλείων Τεχνητής Νοημοσύνης (AI)
Για την ανάπτυξη της εργασίας αξιοποιήθηκε το μοντέλο Τεχνητής Νοημοσύνης Gemini της Google. 
* **Τρόπος αξιοποίησης:** Χρησιμοποιήθηκε ως βοηθός εκμάθησης (tutor) για την κατανόηση της σύνδεσης PHP με MySQL και για οδηγίες παραμετροποίησης δικτύου.
* **Τμήματα κώδικα:** Η σύνταξη των εντολών SQL για την εμφάνιση των δεδομένων και τα CSS styles για το UI γράφτηκαν με την καθοδήγηση του AI.

## ΜΕΡΟΣ Β: Δικτυακή Αρχιτεκτονική

Η τοπολογία σχεδιάστηκε στο Cisco Packet Tracer και είναι κυκλική. Αποτελείται από 5 Routers (μοντέλο 2911), 2 Switches (2960) και 2 PCs. Οι υπολογιστές βρίσκονται σε διαφορετικά υποδίκτυα και είναι τοποθετημένοι σε αντιδιαμετρικά σημεία. Για τη δρομολόγηση των πακέτων υλοποιήθηκε το πρωτόκολλο δυναμικής δρομολόγησης **RIP**.
